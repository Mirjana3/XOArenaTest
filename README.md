# XO Arena - Tic-Tac-Toe Game with Automated Tests

## Opis projekta
XO Arena je web igra **Tic-Tac-Toe** gdje igrač igra protiv AI-a. Projekt uključuje frontend u HTML/CSS/JS i testove napisane u **Selenium + TestNG** za provjeru funkcionalnosti igre.


## Korišteni alati i tehnologije
- **Frontend:** HTML, CSS, JavaScript
- **Testiranje:** Selenium WebDriver, TestNG
- **Build & IDE:** IntelliJ IDEA
- **Verzija kontrole:** Git & GitHub
